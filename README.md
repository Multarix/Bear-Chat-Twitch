# Instructions
1. [Create  a new overlay](https://streamelements.com/dashboard/overlays) on stream elements.
2. Select 1080p resolution
3. Click the `+` and go to `Static/Custom` -> `Custom Widget`
4. Under `Position, size and style`:
	- Width: `1920`
	- Height: `350`
	- Top: `730`
	- Left: `0`

5. Under `Settings`, Open Editor
	- Paste the contents of `index.html` into the `HTML` section
	- Paste the contents of `style.css` into the `CSS` section
	- Paste the contents of `script.js` into the `JS` section
	- Paste the contents of `fields.json` into the `FIELDS` section
6. Optional: Paste the contents of `data.json` into the `DATA` section
7. ???
8. Profit.
